L'implémentation de notre projet nous avons du prendre des décisions sur les différentes technologies que nous pouvions utiliser pour notre projet. Voici nos différents choix:

### Serveur Applicatif Java EE 

Nous avons décidé d'utiliser WildFly version 17.0 car la dernière version n'était pas encore stable. Nous avons créer un docker étant connecter à un serveur Mysql. 

### Application

Notre application est un site web de gestion de ligue de football américain. Nous avons la possibilité de créer des équipes, des joueurs, des arbitres, et des matchs.

Le site se base sur une infrastructure basée sur des pages de liste d'enregistrement. 

Il est nécessaire de se connecter pour voir toute autre page que la page login. Il est aussi demandé d'avoir des droits de niveau 3 pour ajouter des enregistrements.
